# RATIFY External Review Packet — V1

This packet contains two construction candidates:

- progressive primary balance; and
- protected partial fiscal path.

Reviewers should estimate each candidate as an integrated package against the
same current-law baseline. Adding isolated option scores is prohibited. Every
returned table must preserve candidate ID, scenario, fiscal year, unit, method,
source date, uncertainty, and reviewer identity.

Validate the packet and regenerate its file-integrity lock:

```powershell
python review-packets/v1/validate.py
```

Synthetic sensitivity outputs are background diagnostics only. They are not
inputs a reviewer must reproduce and may not be presented as an independent
score.
