# Consensus Rules

This experiment holds national support constant while redistributing it across
states. It compares:

- national popular majority;
- majority of states;
- Electoral College winner-take-all weighting;
- popular-plus-state double majority; and
- a 55%-plus-30-states consensus threshold.

Run:

```powershell
python experiments/consensus-rules/simulate.py
```

The model uses electoral allocations from the 2024 cycle and House seats as a
transparent population proxy. It is an aggregation stress test, not public
opinion, a forecast, or a constitutional recommendation.
