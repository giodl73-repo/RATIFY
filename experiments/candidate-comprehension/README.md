# Candidate Comprehension Stress

This experiment tests recovery of five card facts: marginal payer/exposure,
fiscal target, 2036 debt, correction rule, and unscored status. It does not ask
which package an agent supports.

Synthetic agents are sampled from Voter Lab v1. Neutral, candidate-favorable,
and overloaded presentations run across five seeds and two fatigue levels.
Parameters are heuristic and uncalibrated; results guide card revision only.

Run:

```powershell
python experiments/candidate-comprehension/simulate.py
```
