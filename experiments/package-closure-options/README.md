# Package Closure Options

This experiment applies three explicit closure choices to the retained
broad/shared sensitivity results:

1. raise the narrow VAT from 5% to 11.5%, with the household credit scaling at
   the same rate;
2. add a 3-point high-threshold AGI surtax and full-rate current taxation of
   foreign corporate income; or
3. retain the existing package but choose TAXLANE's partial fiscal path.

These are construction diagnostics, not scores or voter preferences. Run:

```powershell
python experiments/package-closure-options/simulate.py
```
