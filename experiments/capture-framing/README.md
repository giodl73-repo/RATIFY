# Capture and Framing

This synthetic experiment holds four fiscally closed renewal packages constant
and changes only declared presentation assumptions:

- outcome-first full evidence;
- incumbent branding;
- paid advocacy with funding hidden;
- identical advocacy with funding disclosed;
- weaker evidence; and
- ballot overload.

Run:

```powershell
python experiments/capture-framing/simulate.py
```

The parameters are hypotheses, not measured persuasion effects. The experiment
tests whether RATIFY can expose assumption sensitivity and enforce fiscal
closure before using empirical estimates later.
