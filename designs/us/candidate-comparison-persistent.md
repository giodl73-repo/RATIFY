# Persistent Candidate Comparison Interface

The interface keeps one compact difference strip visible while either detailed
card is read:

| Always visible | Progressive primary balance | Protected partial path |
|---|---|---|
| Added cost falls mainly on | Higher-income filers and multinational stakeholders | Future taxpayers and service users through debt and interest |
| 2036 target | Primary balance | Partial adjustment |
| 2036 debt comparison | 107.6% of GDP | 113.6% of GDP |

Above the strip, a fixed banner states: **Neither candidate has an independent
score. Neither can automatically cut protected services.**

When a reader opens the second detailed card, a common-facts refresh restates:

- the underlying protected policy package is identical;
- only the funding/debt choice changes;
- the progressive automatic corridor affects only its surtax;
- the partial path has no automatic tax or service change; and
- anything outside either rule requires new legislation and ratification.

The strip uses equal width, type, color, order, precision, and interaction for
both candidates. Candidate order is randomized in testing. The refresh informs;
it never blocks navigation, preference expression, or voting.
