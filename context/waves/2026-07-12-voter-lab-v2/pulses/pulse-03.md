# Pulse 03: Order and Information Decay

## Result

Added randomized two-card order and a second-card penalty scaled by each agent's
fatigue sensitivity. V1 remains the bridge control with independent traits and
no order decay.

V2 produces a 4.0–4.4 point second-card decline across neutral and layered
presentations. Because order is randomized, candidate-wide accuracy remains
symmetric; individual information quality still depends on position.
