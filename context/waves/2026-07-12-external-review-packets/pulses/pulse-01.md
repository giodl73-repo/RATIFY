# Pulse 01: Candidate Scoring Manifests

## Result

Created one symmetric manifest for two candidates using a common baseline,
2026–2036 window, policy input set, scenarios, annual fiscal fields,
distribution dimensions, interaction matrix, return formats, and comparison
rules. The only candidate-specific fields are target, marginal policy,
correction corridor, and outside-corridor action.

The validator rejects asymmetric candidate fields and writes SHA-256 locks for
all seven common machine inputs.
