# Wave: Package Closure Options

## Objective

Turn the returned broad/shared gap into a small set of complete, intelligible
choices that fund the same policy package without administrative-savings plugs
or hidden reductions in rights and continuity protections.

## Candidate paths

1. **Broader contribution:** adjust the VAT rate or base with a correspondingly
   rescored household credit.
2. **Progressive closure:** add enumerated high-income or corporate instruments
   already present in the variant evidence set.
3. **Lower fiscal target:** retain the policy package but adopt the partial
   fiscal path and disclose its higher debt trajectory.

## Pulses

| Pulse | Purpose | Status |
|---|---|---|
| 01 | Specify closure arithmetic and invariants | complete |
| 02 | Build three complete candidate packages | complete |
| 03 | Stress-test household, service, and regional effects | active |
| 04 | Select packages eligible for joint scoring and voter comprehension tests | pending |

## Decision gate

Can voters compare a small number of honestly funded packages without confusing
package construction with the final binary ratification ballot?
