# Voter Lab Cross-Cutting Program

Voter Lab is continuous infrastructure across RATIFY waves. It does not replace
the active bounded wave or create permission to expand its substantive scope.

Every experiment that models people must either:

- use the current Voter Lab version; or
- record why a narrower local model is necessary and run a bridge comparison.

## Current version

- `ratify-voter-lab-v1`
- synthetic prior, not empirically calibrated
- 11 mixture components
- 14 continuous traits

## Promotion gate

A new version requires invariants, same-scenario bridge runs, sensitivity
results, calibration evidence where used, held-out evaluation where possible,
role review, and an explicit statement of what improved and what regressed.
