# Pulse 03: Voter Lab Gate

## Decision

Do not run substantive package-preference simulations yet.

The matrix is useful for package construction, but it deliberately excludes
dynamic net interest, macroeconomic feedback, annual incidence, and nonfiscal
service results. Voter Lab may test comprehension and framing of uncertainty;
it may not describe synthetic support for this incomplete challenger.

The next package version must name how it closes the fiscal gap before it is a
fair Yes/No alternative to current law.
